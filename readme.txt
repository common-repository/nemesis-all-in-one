The Nemesis All-in-One addon for Elementor is the only one plugin for building blog post pages. 

== Description ==

[youtube https://www.youtube.com/watch?v=juxHFDNJx30]

**Nemesis All-in-One** improves user experience and gives them a chance to create their content with an All in One posts block widget for Elementor. 
Instead of using an add-on with many widgets, you only have one that allows you to create many different designs as long as you have imagination. Leave everything else to us.
If you are still not satisfied with this add-on, take a look at our <a href="https://themeforest.net/item/nemesis-news-magazine-wordpress-theme/34161544">Nemesis - News & Magazine WordPress Theme</a> at Themeforest.
There are many many other widgets as Sliders, Load More Posts etc...


== Installation ==

1. Upload the plugin files to the `/wp-content/plugins/` directory, or install the plugin through the WordPress plugins screen directly.
2. Activate the plugin through the 'Plugins' screen in WordPress.
3. Navigate to Nemesis -> Welcome -> Watch the video to learn how to create your content


== UPDATING ==

Automatic updates should work like a charm; as always though, ensure you backup your site just in case.


== GET A COMPLETE WEBSITE IN JUST 3 STEPS ==

1. Install and activate Nemesis All in One plugin
2. Add a new Page with Elementor Page Builder
3. Start to create newspaper content.
